# nsheaps/.org

Organization-level shared resources for the nsheaps GitHub organization.

## Contents

### `linters/`

Shared linter configurations referenced by repos across the org.

### `labels/`

GitHub label definitions for org-wide label sync (planned).

## Usage

Repos can reference shared configs from this repo using appropriate mechanisms
(e.g., npm packages, remote config references, direct downloads in CI).
